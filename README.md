# UHST v18 – Monte Carlo Blind Prediction Test

This repository links to the public Zenodo release of the **UHST v18** analysis package.

## Zenodo record
DOI: **10.5281/zenodo.18889324**

Record:
https://zenodo.org/records/18889324

## Summary

This release contains a reproducible analysis package for the UHST v18 study of galactic rotation curves using the SPARC dataset.

Main features:
- 175 galaxies
- 3389 rotation-curve measurements
- 100 Monte Carlo cross-validation splits
- blind train/test evaluation on the galaxy level
- manuscript PDF and LaTeX source
- reproducibility scripts
- publication figures

## Main result

The MCCV experiment indicates that the outer-regime parameters remain stable across unseen galaxy splits, while the inner transition parameters remain weakly constrained.
